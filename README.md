# DevRoast

Paste your code. Get roasted.

DevRoast is a code roasting app that gives your code a brutally honest review. Paste any snippet, toggle **roast mode** for maximum sarcasm, and get a score from 0 to 10 along with a detailed analysis of everything wrong (and occasionally right) with your code.

## Features

- **Code input** -- Paste or type any code snippet and submit it for review.
- **Roast mode** -- Toggle between a polite review and full-on sarcasm.
- **Score & analysis** -- Get a score out of 10 with a breakdown of issues, suggested improvements, and a rewritten version of your code.
- **Diff view** -- See exactly what changed between your original code and the improved version.
- **Shame leaderboard** -- The worst code on the internet, ranked by score. See how your submission stacks up against the hall of shame.
- **Shareable results** -- Each roast generates an OG image you can share.

## Running locally

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).
